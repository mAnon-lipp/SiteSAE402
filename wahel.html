Economy & Customer Management

Immersive Retail Experience

I transformed the base game into a true retail simulation. My work focused on managing dynamic customer queues, implementing an in-game economy with a decoration shop, and enhancing immersion through spatial audio.

My Technical Contributions

Task Management, Customer Queues, Economy & Audio

1. Dynamic To-Do List & Tutorial

Creation of an interactive checklist to guide the player. The system dynamically reorders tasks and validates steps (checkup) to ensure a smooth learning process of the mechanics before the real service begins. 

// src/main.js
function updateTutorialUI() {
    let msg = "TO DO:\n\n";

    if (tutorialStep === 1) msg += "> Buy Broom & Dustpan (Menu Y)\n";
    else if (tutorialStep > 1) msg += "[x] Floor cleaned\n";
    
    // ...
    
    if (tutorialStep === 5) msg += "> Serve coffee to customer\n";
    else if (tutorialStep > 5) msg += "[x] Customer served\n";

    tutorialText.setAttribute('value', msg);
    if (tutorialStep === 7) startGameMode();
}

2. Dynamic Customer Queues

Development of a customer management system. The queue and the customer's position at the counter dynamically adapt to the player's real-world position. I also added a patience limit bar and a specific leaving animation if the customer gets angry from waiting too long. 

// src/main.js
customer._patienceTimer = setInterval(function () {
    customer._patience -= decrementPerTick;
    
    // Update visual patience bar width based on %
    var pct = customer._patience / 100;
    customer._patienceBar.setAttribute('width', pct * 1.0);

    // Out of patience!
    if (customer._patience <= 0) {
        customerLeaveAngry(customer);
    }
}, 200);

3. In-Game Economy & Decoration Shop

Implementation of the cash register and monetization system. Once an order (coffee or water) is handed to the customer, they pay, and the money must be put into the register. This earned cash can then be spent in an in-game shop to buy decorations for your virtual coffee shop. 

// src/main.js
if (price > 0 && totalEarnings < price) {
    showARNotification('Not enough money! Need $' + price);
} else {
    if (price > 0) totalEarnings -= price; // Deduct money
    
    // Update wallet display in VR store
    let wallet = document.getElementById('shop-wallet');
    if (wallet) wallet.setAttribute('value', '$' + totalEarnings);

    spawnObject(type, color, model, scale); // Spawn item
}
