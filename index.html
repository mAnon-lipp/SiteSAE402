Code excerpts and description of the main implemented functionalities.

1. "Spring Cleaning" Phase

Before opening, the user must clean their space. This mechanic gamifies technical calibration by using hand or broom tracking to erase procedural stains. 

// Excerpt from src/main.js - Cleaning System
            function checkCleaning() {
            if (!grabbed || !currentGrabbedEl) return;
            if (!currentGrabbedEl.getAttribute('gltf-model').includes('Broom')) return;
            stains.forEach((stainObj, index) => {
                const dist = new THREE.Vector2(broomPos.x, broomPos.z).distanceTo(new THREE.Vector2(stainPos.x, stainPos.z));
                if (dist < 0.4 && Math.abs(broomPos.y - stainPos.y) < 0.5) {
                stainObj.health -= 5; // Progressive stain reduction
                stainObj.el.setAttribute('opacity', stainObj.health / 100);
                }
            });
            }

2. Machine Interaction and Object Spawning

Gameplay relies on direct interaction: point at the machine and trigger preparation. The system manages audio feedback and cup instantiation in 3D space. 

// Excerpt from src/main.js - Business Logic
            function handleCoffeeMachineClick(machineEntity) {
            if (coffeeMachineLock) return;
            coffeeMachineLock = true;
            coffeeAudio.play(); // Spatialised audio feedback
            setTimeout(() => {
                const cup = document.createElement('a-entity');
                cup.setAttribute('gltf-model', 'url(models/Coffeecup.glb)');
                cup.setAttribute('dynamic-body', 'mass:0.3'); // Physics activation
                sceneEl.appendChild(cup);
                coffeeMachineLock = false;
            }, 1500);
            }

3. Physical Manipulation and Throwing (Grabbing)

The grab system enables natural object interaction. To ensure smooth movement, physics is temporarily modified (Kinematic mode) before applying impulse during throwing. 

// Excerpt from src/components/hand-grab.js - Grab Management
            this.el.addEventListener('grab-start', () => {
            this.isGrabbed = true;
            if (this.el.body) {
                this.el.body.mass = 0; // Gravity cancellation for grabbing
                this.el.body.type = 2; // Switch to KINEMATIC mode (follows hand)
                this.el.body.collisionResponse = false;
            }
            });

4. Spatial Analysis and Hit-Test

To anchor virtual furniture (counter, machines) on real user furniture, the project uses WebXR API to detect intersections between controller rays and the real world. 

// Excerpt from src/main.js - Hit-Test API
            if (hitTestSource) {
                const hits = frame.getHitTestResults(hitTestSource);
                if (hits.length > 0) {
                const pose = hits[0].getPose(xrRefSpace);
                cursorEl.object3D.position.set(pose.transform.position.x,pose.transform.position.y,pose.transform.position.z);
                cursorEl.object3D.visible = true;
                }
            }
