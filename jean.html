UX Gamification, API Integration & Spatial Interaction

1. Interactive Tutorial

The `TutorialSystem` manages player onboarding through a spatial checklist. It sequentially updates UI panels, ensuring players understand core mechanics before the game rush. 

// src/modules/story.js
completeTask(taskId) {
    if (this.isCompleted) return;
    const taskIndex = this.tasks.findIndex(t => t.id === taskId);
    if (taskIndex !== -1 && !this.tasks[taskIndex].completed) {
        this.tasks[taskIndex].completed = true;
        this.updateUI(); // Updates spatial UI
    }
}
To do Panel

2. Dynamic Difficulty

The game scales through modes. `DarkKitchenSystem` adjust items and scaling. High difficulty introduces new items like donuts and reduces time limits. 

// src/modules/story.js
generateOrder() {
    let possibleItems = ['coffee'];
    if (this.difficulty === 'hard') {
        possibleItems.push('donut');
        this.timeLimit -= 10;
    }
}
Order Mode Activated

3. Leaderboard

Integrated Supabase API to fetch and render global high scores in the VR environment, driving competition. 

// src/modules/score.js
async getLeaderboard(limit = 10) {
    const { data } = await supabase
        .from('leaderboard')
        .select('*')
        .order('score', { ascending: false })
        .limit(limit);
    return data;
}

4. Audio Player

Created `SpeakerPlaylist` for immersive audio. Players use the 'B' button to toggle a music UI and change custom tracks.
